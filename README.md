## Prerequisites

Let's use the sdcc toolchain for STM8.

```
$ sudo apt-get install sdcc
```

To upload the program to the target processor, the
[stm8flash](https://github.com/vdudouyt/stm8flash) will be used. After download, just:

```
$ make
$ sudo make install
```
